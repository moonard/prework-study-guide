HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • CSS changes need to be linked via the HTML file to be seen

CSS

  • A margin indiciates how much space we want around the outside of an element
  • A padding indicates how much space we want inside an element
  • A period before the selector indicates that all of those elements will be affected
  • Multiple elements being affected, such as h1 and h2, should be separated by commas

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A adds the changes from our working directory to the staging area of Git
  • git commit -m "comment here" commits to our repository
  • git pull origin main makes sure your current branch is up to date with what is in the repository
  • git push origin feature/branchhere makes a PR to the repository

JavaScript

  • CSS is in the head, JS is in the body
  • Lines are separated by semicolons
  • Arrays can hold strings, numbers, or Boolean values
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
